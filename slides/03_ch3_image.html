Ch.3 画像処理入門

画像の正体：数値の2次元配列

「画像を見る」＝「数値の表を見る」
手書き数字「0」の8×8ピクセル（値：0〜16）
値が大きい（16に近い）= 明るい（白）
値が小さい（0に近い）= 暗い（黒） 
from sklearn.datasets import load_digits
import numpy as np

digits = load_digits()

# 1枚目の画像（数字「0」）
img = digits.images[0]
print(img.shape)
# → (8, 8)  ← 8×8ピクセル

print(img.astype(int))
# → 数値の配列が表示される

# 画像として表示
plt.imshow(img, cmap='gray_r')
plt.show()
🔑 画像 ＝ 数値の2次元配列
だから NumPy で計算・統計処理できる！

グレースケールとカラー（RGB）の違い

🔲 グレースケール

  • 1チャンネル（明るさのみ）
  • 配列の形：(縦, 横)
  • 値の範囲：0〜255
  • 色情報なし

🌈 カラー（RGB）

  • 3チャンネル（赤・緑・青）
  • 配列の形：(縦, 横, 3)
  • 各チャンネルの値：0〜255
  • 3つの数値の組み合わせで色を表現
グレースケール変換とは？
色情報（RGB）を捨てて、明るさだけを残す処理 
from PIL import Image

# カラー画像を読み込み
img = Image.open('photo.jpg')
print(img.mode)  # → 'RGB'

# グレースケールに変換
img_gray = img.convert('L')
print(img_gray.mode)  # → 'L'
💡 グレースケール変換後は
「赤いリンゴ」と「緑のリンゴ」が
同じ明るさなら区別できなくなる

Pillow（PIL）の基本操作

numpy array ↔ PIL Image の変換が基本
numpy array
(数値の配列)
→ Image.fromarray()
← np.array()
PIL Image
(画像オブジェクト)
→
resize()

convert()
→
処理後の
PIL Image
# numpy array → PIL Image
arr_scaled = (digits.images[0] / 16.0 * 255).astype(np.uint8)
img = Image.fromarray(arr_scaled, mode='L')

# リサイズ（8×8 → 80×80）
img_large = img.resize((80, 80), Image.NEAREST)

# PIL Image → numpy array（元に戻す）
arr_back = np.array(img_large)
print(arr_back.shape)  # → (80, 80)

画像データの統計分析

画像 ＝ 数値 → 統計量を計算できる → 機械学習の特徴量として使える！
  • 平均ピクセル値 → 画像の「明るさ」を数値化
  • 標準偏差 → 明暗のばらつき（コントラスト）
  • 最大・最小 → 最も明るい/暗い箇所
  • 0超のピクセル数 → 「描かれている量」
💡 実務では画像の特徴量（平均輝度・エッジ量など）を計算して機械学習モデルに入力します 
# PIL Image → numpy array
arr = np.array(img_large)

# 統計量の計算
print('明るさ（平均）:', arr.mean())
print('コントラスト:', arr.std())
print('最大値:', arr.max())
print('最小値:', arr.min())

# 描かれているピクセルの割合
drawn = (arr > 0).mean() * 100
print(f'描画率: {drawn:.1f}%')
数字「1」は細くてピクセルが少ない
→ 平均が低い（暗い）
