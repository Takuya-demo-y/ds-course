## 各チャプター詳細

### オリエンテーション（20分・座学）

**目的：** 1日の流れとゴールの共有、生成AI活用ルールの説明

- データサイエンティストの仕事の全体像（分析→可視化→モデル→解釈）
- 本日扱うデータ・課題の紹介
- 生成AI活用ルールの説明（← 最初に全員で合意する）

#### 生成AI活用ガイドライン（コース全体共通）

| カテゴリ | AIに任せてよいこと | 自分で理解すること |
| --- | --- | --- |
| コード生成 | ボイラープレート（import、ファイル読み込み）、ライブラリのAPI使い方 | ロジックの流れ、パラメータの意味 |
| エラー対応 | エラーメッセージの翻訳・修正案 | なぜそのエラーが起きたか |
| 結果解釈 | グラフのタイトル・軸ラベル案 | 数値の意味、ビジネス上の示唆 |

#### プロンプトテンプレート（演習で配布）

```
【目的】〇〇のデータを使って△△を分析したい
【使うライブラリ】pandas, matplotlib（Pillow/scikit-learnなど）
【制約】Python 3.8、外部ライブラリは使わない
【やりたいこと】〇〇列を基準にグループ集計して、棒グラフで表示したい
【わからない点】groupbyの書き方
```

### Ch.1 データ読み込み・集計（60分 ｜ 座学20分 ＋ 演習40分）

**使用ライブラリ：** `pandas` `numpy`

#### 座学（20分）

- CSVの読み込み（`pd.read_csv`）、基本確認（`head` `info` `describe`）
- 欠損値の確認・処理（`isnull` `fillna` `dropna`）
- グループ集計（`groupby`）、ソート（`sort_values`）
- 「DSがよくやる作業」として実務イメージを紹介

#### 演習（40分）

**題材：** タイタニック乗客データ（生存/非生存）

| ステップ | 内容 |
| --- | --- |
| ステップ1（AIで生成） | 読み込み・基本確認コードをAIに生成させ、実行確認 |
| ステップ2（自力） | 「性別・年齢層ごとの生存率」をgroupbyで集計する |
| ステップ3（発展） | 欠損値の処理方針を自分で考えて実装 |

#### 演習でのAI活用ポイント

- ✅ AIに任せる：`pd.read_csv` のオプション（文字コード指定など）
- ✅ AIに任せる：エラー「KeyError: '列名'」の修正
- ❌ 自分でやる：どの列を集計すべきか、なぜその集計が意味あるか

### Ch.2 データ可視化・EDA（60分 ｜ 座学20分 ＋ 演習40分）

**使用ライブラリ：** `matplotlib` `seaborn` `japanize-matplotlib`

#### 座学（20分）

- EDA（探索的データ分析）とは何か、なぜやるか
- ヒストグラム（`hist`）、棒グラフ（`bar`）、散布図（`scatter`）
- 箱ひげ図（`boxplot`）、ヒートマップ（`heatmap`）
- 相関係数（`corr`）の見方

#### 演習（40分）

**題材：** Ch.1の続き（タイタニックデータ）

| ステップ | 内容 |
| --- | --- |
| ステップ1（AIで生成） | 相関ヒートマップのコードをAIで生成・実行 |
| ステップ2（自力） | 生存率と各変数の関係を可視化して「仮説」を1つ言語化する |
| ステップ3（発展） | 日本語ラベル付きグラフを完成させる（japanize-matplotlib） |

#### 演習でのAI活用ポイント

- ✅ AIに任せる：`seaborn.heatmap` の引数（`annot=True` など）
- ✅ AIに任せる：グラフの色・サイズ調整
- ❌ 自分でやる：「このグラフから何が読み取れるか」の解釈

### Ch.3 画像処理入門（60分 ｜ 座学20分 ＋ 演習40分）

**使用ライブラリ：** `Pillow (PIL)` `numpy` `matplotlib`

#### 座学（20分）

- 画像データの正体（ピクセル値の配列）を `numpy.array` で確認
- 基本操作：読み込み（`Image.open`）、リサイズ（`resize`）、保存（`save`）
- 色変換：グレースケール（`convert('L')`）
- フィルタ処理：ぼかし（`ImageFilter.BLUR`）、輪郭抽出（`CONTOUR`）
- 実務例：製造業の外観検査、医療画像などのユースケース紹介

#### 演習（40分）

**題材：** 提供する数枚のサンプル画像（風景・製品画像）

| ステップ | 内容 |
| --- | --- |
| ステップ1（AIで生成） | 画像読み込み→グレースケール変換→表示のコードをAIで生成 |
| ステップ2（自力） | ピクセル値の平均・最大・最小をnumpyで計算して「明るさ」を数値化 |
| ステップ3（発展） | 複数フィルタを試して、処理前後を並列表示（matplotlib subplot） |

#### 演習でのAI活用ポイント

- ✅ AIに任せる：`ImageFilter` のオプション一覧の確認
- ✅ AIに任せる：`subplot` の並べ方
- ❌ 自分でやる：「グレースケール変換すると何の情報が失われるか」の理解

### Ch.4 機械学習モデル構築（60分 ｜ 座学20分 ＋ 演習40分）

**使用ライブラリ：** `scikit-learn` `pandas` `matplotlib`

#### 座学（20分）

- 教師あり学習のワークフロー（前処理→学習→予測→評価）の全体像を5分で
- データ分割：`train_test_split`
- 前処理：`StandardScaler`（なぜスケーリングが必要か）
- モデル：ランダムフォレスト（`RandomForestClassifier`）1種に絞る
- 評価：正解率（Accuracy）、混同行列

#### 演習（40分）

**題材：** タイタニックデータで生存予測（Ch.1〜2の続き）

| ステップ | 内容 |
| --- | --- |
| ステップ1（AIで生成） | 前処理〜モデル学習〜正解率表示のパイプラインをAIで生成 |
| ステップ2（自力） | 混同行列を見て「誤検知vs見逃し」どちらが多いか分析 |
| ステップ3（発展） | `n_estimators` を変えて精度の変化を確認 |

#### 演習でのAI活用ポイント

- ✅ AIに任せる：one-hot encoding のコード、`confusion_matrix` の表示
- ❌ 自分でやる：「このモデルをどう改善するか」の仮説立て
- ❌ 自分でやる：「正解率が高い＝良いモデル」とは限らない理由の言語化

**題材（別データセットを用意）：** アイリス（花の品種分類）やワインデータなど

#### 課題シート（ステップ別に配布）

| ステップ | タスク | AIの活用可否 |

| 3 | 最低2種の可視化と「発見」の言語化 | AIで生成OK・解釈は自力 |
| 4 | モデル構築と評価 | AIで生成OK |
