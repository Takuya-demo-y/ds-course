### Ch.1 データ読み込み・集計（9:20〜10:20）

**座学のポイント（20分）：**
- `df.head()` → `df.info()` → `df.describe()` の順番に意味がある（形→型→統計）
- 欠損値は「削除」か「補完」かはデータ量と欠損の意味次第（正解はない）
- `groupby` は「業務で毎日使う」と伝えるとイメージしやすい

**演習でよくあるつまずき：**
- `KeyError` → 列名のスペルミス（`df.columns` で確認を促す）
- `groupby` の結果が多次元になる → `.reset_index()` を使うよう案内

---

### Ch.2 データ可視化・EDA（10:30〜11:30）

**座学のポイント（20分）：**
- EDA は「仮説を立てるための作業」と位置づける
- グラフの種類は「何を知りたいか」で決まる（分布→ヒストグラム、比較→箱ひげ図）
- 相関係数は「-1〜1の範囲」「因果ではない」を強調

**演習の進め方：**
- 問3の「仮説の言語化」は時間がかかる受講者が多い。3分考えて書けなければ隣の人と相談してOK
- `japanize_matplotlib` を忘れると日本語が文字化けする。冒頭でインポートを確認

---

### Ch.3 画像処理入門（11:30〜12:30）

**座学のポイント（20分）：**
- 「画像 = 数値の配列」という概念が最大のポイント。`print(img_array.astype(int))` で実際の数値を見せる
- グレースケール変換で「色の情報が消える」→ 色を使った識別ができなくなる
- 実務ユースケース（製造業の外観検査など）と結びつけてモチベーションを上げる

**演習でよくあるつまずき：**
- `mode='L'` を忘れると RGB 画像になる
- `Image.NEAREST` が見つからない → `from PIL import Image` の確認
- フィルタ名が大文字なので大文字小文字のタイポが多い

---

### Ch.4 機械学習モデル構築（13:30〜14:30）

**座学のポイント（20分）：**
- `scaler.fit` は訓練データのみ。これが「データリーク」防止の最重要ポイント
- ランダムフォレストの仕組みは「多数決」で覚えてもらえれば十分
- 混同行列は「FP（誤検知）とFN（見逃し）のどちらが問題か」はビジネス次第と強調

**演習でよくあるつまずき：**
- `X_test_scaled` に `fit_transform` を使ってしまう（`transform` のみと案内）
- `classification_report` の出力フォーマットが崩れる → フォントをmonospaceにするよう案内
