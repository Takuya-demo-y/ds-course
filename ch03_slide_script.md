### スライド 03-04｜【概念①】コンピュータに「絵」は見えない

**コンピュータは画像を「数値の格子」として見ている**

**あなたが見るもの vs コンピュータが見るもの**

```
あなたの目               コンピュータの目

  ┌──────┐              0  0  5 13  9  1  0  0
  │  ○  │              0  0 13 15 10 15  5  0
  │ ○  ○│              0  3 15  2  0 11  8  0
  │ ○  ○│              0  4 12  0  0  8  8  0
  │  ○  │              0  5  8  0  0  9  8  0
  └──────┘              0  4 11  0  1 12  7  0
    「0だ」              0  2 14  5 10 12  0  0
                        0  0  6 13 10  0  0  0

                        「64個の数値だ」
```

**数値の意味**
- 値が大きい（最大16）→ 明るい（白）
- 値が小さい（0）→ 暗い（黒）
- 8 × 8 ＝ **64個の数値** ＝ 1枚の画像

**なぜこれが重要か**
> 画像が「数値の配列」なら、統計や機械学習がそのまま使える。  
> 平均も計算できる。距離も計算できる。異常スコアも出せる。

**補足**
> LEDパネルに近づいて見ると小さな点の集まりなのと同じ。コンピュータはいつも「点の明るさ」で画像を認識している。

**スライドノート（講師向け）**

### スライド 03-05｜【概念②】「正常を定義する」とはどういうことか

**「正常」の基準は、正常なサンプルをたくさん集めると自然に浮かび上がる**

**具体的なイメージ**

```
178枚の「正常な0」を透かして重ねると…

  枚目1:  0  0  5 13  9  1  0  0
  枚目2:  0  1  6 12  8  2  0  0
  枚目3:  0  0  4 14  9  0  0  0
     ↓ 178枚を平均する
  平均:   0  0  5 13  9  1  0  0 ← 「典型的な0の顔」
```

**日常のアナロジー**

> 「標準的な日本人の顔」を作りたい → 多くの顔写真を半透明に重ねて平均すると、「典型的な顔」が浮かぶ。特徴的な個人の顔ほど平均から外れる。

**異常検知への接続**

```
正常な0 → 平均プロファイルと「似ている」
異常な0 → 平均プロファイルと「似ていない」（傾いている・薄い・変形）

「似ていない度合い」 = 異常スコア
```

**発問①（予測発問）**
> 💬 「178枚の0を平均した画像を表示したら、どんな見た目になると思いますか？」  
> ※ 10秒考えてもらう → 回答を聞く → コードで確認する

**スライドノート（講師向け）**
> 「正常を定義する」という概念がこのスライドの核心。コードを見せる前に、「なぜ平均が正常の基準になるのか」が腑に落ちているかを確認してから次へ。発問①の回答を受けてからデモを実行する。

### スライド 03-06｜【概念③】「正常から遠い」 ＝ 異常スコア

**距離で「異常っぽさ」を数値化する**

**直感的なイメージ**

```
正常プロファイル（平均の0）から…

 近い → ほとんど同じ顔の0 → スコアが低い → 正常
 遠い → 全然違う顔の0   → スコアが高い → 異常
         ↑
      傾いている・薄い・ぐちゃっとしている
```

**距離の測り方（Zスコア）**

> 「平均から何ステップ離れているか」を各ピクセルで計算し、64個の平均を取る

| 具体例 | ピクセル値 | 平均値 | 差 | 異常度 |
|--------|----------|--------|-----|--------|
| 典型的な0のピクセル | 13 | 13.1 | 0.1 | 小 |
| 変な0のピクセル | 0 | 13.1 | 13.1 | 大 |

**重要な考え方**
> スコアの高い・低いは相対的なもの。「他のサンプルと比べて外れているか」で決まる。

**発問②（理由発問）**
> 💬 「スコアが最も高かった（最も変な0）の画像を見て、なぜ異常と判定されたと思いますか？」  
> ※ 画像を表示した後に問いかける

**スライドノート（講師向け）**
> 「Zスコア」という言葉は出さなくていい。「平均から離れているほど異常」という直感だけ伝える。数式の説明は不要。コードを動かして「変な0がランクインした」という事実で理解させる。

### スライド 03-07｜【概念④】IsolationForestの直感的な理解

**「孤立しやすいデータ」ほど異常**

**考え方のアナロジー**

```
人が100人いる部屋を「分割」で絞り込む

 質問1:「身長が170cm以上ですか？」→ 50人が残る
 質問2:「30歳以上ですか？」→ 25人が残る
 ：
 ：
```

```
正常なデータ（多数派）       異常なデータ（少数派）
──────────────────────       ─────────────────────
 たくさんの質問が必要         少ない質問で孤立できる
 切り離しにくい             あっさり切り離される
 → スコアが低い（正常）       → スコアが高い（異常）
```

**Zスコアとの違い**

| 比較 | Zスコア（手動） | IsolationForest |
|------|--------------|----------------|
| 方法 | 平均からの距離を計算 | 孤立しやすさで判定 |
| 良いところ | 直感的・説明しやすい | 複雑なパターンも検出できる |
| コード量 | 自分で計算式を書く | 3行で動く |

**今日使う設定**

> `contamination=0.05` ＝「全体の約5%が異常」という前提  
> この数字はコードではなく**業務知識**で決める → これがデータサイエンティストの仕事

**スライドノート（講師向け）**
> 「IsolationForestのアルゴリズム詳細は説明しない。『孤立しやすいデータ＝異常』という直感だけ掴んでもらえれば十分。コードは3行。難しいことは中でやってもらう」と話す。`contamination`の設定はDSの仕事という点を強調する。

### スライド 03-08｜コードで確認する（デモ）

**概念を確認する ― 4ステップをまとめて動かす**

> ここから手を動かします。コードの意味はすでに理解済みです。  
> 「このコードを実行すると、さっきの概念が動く」と思いながら実行してください。

**STEP 1：画像と数値が同じものだと確認する**

```python
from sklearn.datasets import load_digits
import numpy as np
import matplotlib.pyplot as plt

digits = load_digits()
print(digits.images[0])                      # 数値として見る
plt.imshow(digits.images[0], cmap="gray_r")  # 画像として見る
plt.show()
```

**STEP 2：正常な0を取り出し、平均プロファイルを作る**

```python
X = digits.data    # 1797枚 × 64次元
y = digits.target

X_zeros      = X[y == 0]                # 数字「0」だけ取り出す
mean_profile = X_zeros.mean(axis=0)    # 178枚の平均 → 「典型的な0」

plt.imshow(mean_profile.reshape(8, 8), cmap="gray_r")
plt.title("正常の基準（平均プロファイル）")
plt.show()
```

**STEP 3：異常スコアを計算してランキングを出す**

```python
std_profile = X_zeros.std(axis=0)

def calc_anomaly_score(sample, mean, std):
    z = np.abs((sample - mean) / (std + 1e-8))
    return z.mean()

scores   = np.array([calc_anomaly_score(x, mean_profile, std_profile)
                     for x in X_zeros])
top5_idx = np.argsort(scores)[::-1][:5]  # スコアが高い順トップ5
```

**STEP 4：IsolationForestで自動検知する**

```python
from sklearn.ensemble import IsolationForest

clf         = IsolationForest(contamination=0.05, random_state=42)
clf.fit(X_zeros)
predictions = clf.predict(X_zeros)  # 1=正常、-1=異常

print("異常と判定された件数:", (predictions == -1).sum())
```

**補足**
> コードを覚える必要はない。「このコードを動かすと何が起きるか」を理解することが目的。

## 付録：コード全体まとめ（ノートブック作成時に使用）

> ※ スライドに分散したコードを「座学デモ用」として1か所にまとめたもの。  
> ノートブック `ch3_anomaly_detection.ipynb` の講師デモセルに使用する。  
> **スライドでは概念の説明をして、このコードを実行するのがSTEP 03-08の流れ。**

```python
# ========================================
# Ch.3 座学デモ：異常検知入門
# ========================================
from sklearn.datasets import load_digits
from sklearn.ensemble import IsolationForest
import numpy as np
import matplotlib.pyplot as plt

# ─ STEP 1: 画像 = 数値の格子（概念①の確認）─
digits = load_digits()
print(digits.images[0])                      # 数値として見る
plt.imshow(digits.images[0], cmap="gray_r")  # 画像として見る
plt.title(f"ラベル: {digits.target[0]}")
plt.show()

# ─ STEP 2: 正常プロファイルを作る（概念②の確認）─
X = digits.data
y = digits.target
X_zeros      = X[y == 0]
mean_profile = X_zeros.mean(axis=0)
std_profile  = X_zeros.std(axis=0)

plt.imshow(mean_profile.reshape(8, 8), cmap="gray_r")
plt.title("正常の基準（数字0の平均プロファイル）")
plt.colorbar()
plt.show()

# ─ STEP 3: 異常スコアを計算（概念③の確認）─
def calc_anomaly_score(sample, mean, std):
    z = np.abs((sample - mean) / (std + 1e-8))
    return z.mean()

scores   = np.array([calc_anomaly_score(x, mean_profile, std_profile)
                     for x in X_zeros])
top5_idx = np.argsort(scores)[::-1][:5]

fig, axes = plt.subplots(1, 5, figsize=(10, 2))
for i, idx in enumerate(top5_idx):
    axes[i].imshow(X_zeros[idx].reshape(8, 8), cmap="gray_r")
    axes[i].set_title(f"#{i+1}\n{scores[idx]:.2f}")
plt.suptitle("スコアが高い（最も変な0）トップ5")
plt.tight_layout()
plt.show()

# ─ STEP 4: IsolationForestで自動検知（概念④の確認）─
clf         = IsolationForest(contamination=0.05, random_state=42)
clf.fit(X_zeros)
predictions = clf.predict(X_zeros)  # 1=正常、-1=異常

print("異常と判定された件数:", (predictions == -1).sum())

anomaly_idx = np.where(predictions == -1)[0]
fig, axes   = plt.subplots(1, min(5, len(anomaly_idx)), figsize=(10, 2))
for i, idx in enumerate(anomaly_idx[:5]):
    axes[i].imshow(X_zeros[idx].reshape(8, 8), cmap="gray_r")
    axes[i].set_title("異常")
plt.suptitle("IsolationForestが異常と判定したサンプル")
plt.tight_layout()
plt.show()
```
