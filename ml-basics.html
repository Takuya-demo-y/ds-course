1
Supervised Learning
教師あり学習
正解ラベルつきのデータを使ってモデルを訓練し、未知のデータを予測する
💡 たとえで理解する

答えつきの問題集で勉強するイメージ。数学の試験前に「問題→答え」のセットを大量に解くように、AIは「入力データ→正解ラベル」のペアを繰り返し学習し、パターンを習得する。

1
データ収集入力データ（特徴量）と正解ラベルをセットで大量に用意する
2
モデルの訓練モデルが予測 → 正解と比較 → 誤差（損失）を計算 → パラメータを更新する
3
未知データで予測学習済みモデルに新しいデータを入力し、クラスや値を出力する
🟦 クラス分類デモ（2クラス） 待機中
🔵 Class A と 🔴 Class B のラベル付きデータを学習させます。「① 学習開始」を押してください。
代表的なアルゴリズム
線形回帰 ロジスティック回帰 決定木 SVM ランダムフォレスト ニューラルネットワーク
主な使いどころ
  • 迷惑メール（スパム）の自動判定
  • 医療画像から病変を診断
  • 機械翻訳・音声認識
  • 株価・需要予測
2
Unsupervised Learning
教師なし学習
正解ラベルなしのデータから、隠れたパターンや構造を自動で発見する
💡 たとえで理解する

図書館の本を誰かに教わらず自分でジャンル別に整理するイメージ。「これは小説っぽい、これは参考書っぽい」と似たものを集め、自然とグループができあがる。答えがなくてもデータ自体に構造がある。

1
ラベルなしデータを用意正解なしでデータだけを大量に集める
2
パターン・構造を発見データの類似度を分析し、グループ（クラスタ）や圧縮表現を自動形成する
3
結果を活用発見したグループやパターンをビジネスや分析に適用する
🟣 k-Means クラスタリングデモ 待機中
グレーの点はラベルなしデータ。アルゴリズムが自動で3グループを発見します。
代表的なアルゴリズム
k-Means クラスタリング 階層的クラスタリング DBSCAN 主成分分析 (PCA) オートエンコーダー t-SNE
主な使いどころ
  • 顧客を購買傾向でグループ化（セグメンテーション）
  • 不正アクセス・異常取引の検知
  • 膨大なデータを少ない次元に圧縮
  • ニュース記事を話題ごとにまとめる
3
Reinforcement Learning
強化学習
環境と相互作用しながら試行錯誤し、報酬を最大化する行動方策を学ぶ
💡 たとえで理解する

犬のしつけのイメージ。「お手」ができたらご褒美（＋報酬）、噛んだら叱られる（－報酬）を繰り返すうち、エージェント（犬）は「どう行動すれば報酬が最大になるか」を自然に学習する。正解データは不要。

1
状態を観測エージェントが環境の現在の状態（State）を観測する
2
行動を選択・実行価値関数（Q値）に基づき行動を選ぶ。最初はランダム、徐々に最適化される
3
報酬を受け取り更新結果に応じた報酬（Reward）をもとにQ値テーブルを更新し、方策を改善する

代表的なアルゴリズム
Q 学習 DQN（Deep Q-Network） Policy Gradient PPO Actor-Critic AlphaZero
主な使いどころ

  • ロボットの歩行・アーム制御
  • 自動運転の経路計画
  • 広告入札・在庫補充の自動最適化
