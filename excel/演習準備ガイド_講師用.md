# 演習ファイル 事前準備ガイド（講師用）

> **目的：** 受講生が「VBAコードを貼り付ける」手順を省略できるよう、  
> 研修当日の前日までに、VBAコードを埋め込んだ `.xlsm` ファイルを作成します。

---

## なぜ事前準備が必要か

現在の演習フォルダには `.xlsx`（データ）と `.txt`（VBAコード）が別々に入っています。  
受講生が「コードを VBE に貼り付ける」という手順は、初心者には想定外に難しく時間がかかります。  
事前に `.xlsm`（VBAが入ったExcelファイル）を作っておくと、**受講生はファイルを開くだけ**で演習を始められます。

---

## 準備するファイル一覧

| フォルダ | 現在のファイル | 作成するファイル |
|---|---|---|
| `01_デバッグ演習/` | `デバッグ演習.xlsx` + `VBAコード_バグあり.txt` | `デバッグ演習_配布用.xlsm` |
| `02_Function演習/` | `売上評価データ.xlsx` + `VBAコード_テンプレート.txt` | `売上評価データ_配布用.xlsm` |
| `03_シート操作演習/` | `月次売上データ.xlsx` + `VBAコード_テンプレート.txt` | `月次売上データ_配布用.xlsm` |
| `04_総合演習/` | `集計ツール_ベース.xlsx` + `VBAコード_テンプレート.txt` | `集計ツール_配布用.xlsm` |

## 手順（各演習ファイル共通）

### Step 1 ── Excel ファイルを開く

`デバッグ演習.xlsx`（または各演習の `.xlsx`）を Excel で開きます。

### Step 2 ── VBE を開く

```
Alt + F11
```

Visual Basic Editor（VBE）が開きます。

### Step 3 ── 標準モジュールを挿入する

VBE のメニューから：

```
「挿入」→「標準モジュール」
```

左側のプロジェクトツリーに `Module1` が追加されます。

### Step 4 ── コードを貼り付ける

対応する `.txt` ファイルを**テキストエディタ（メモ帳等）で開き**、全選択（`Ctrl + A`）してコピー。  
VBE の `Module1` の編集エリアに貼り付け（`Ctrl + V`）。

| 演習 | 使う .txt ファイル |
|---|---|
| デバッグ演習 | `VBAコード_バグあり.txt` |
| Function演習 | `VBAコード_テンプレート.txt` |
| シート操作演習 | `VBAコード_テンプレート.txt` |
| 総合演習 | `VBAコード_テンプレート.txt` |

### Step 5 ── `.xlsm` として保存する

```
「ファイル」→「名前を付けて保存」
→ ファイルの種類：「Excel マクロ有効ブック (*.xlsm)」
→ ファイル名：デバッグ演習_配布用.xlsm
```

> ⚠️ `.xlsx` のまま保存しようとすると「VBAが失われます」という警告が出ます。  
> 必ず **`.xlsm`** を選択してください。

### Step 6 ── 動作確認

- `Alt + F8` でマクロ一覧を表示
- 対象の Sub が表示されていれば OK
- 一度実行して動作確認する

---

## 受講生への配布時の注意

- **配布するのは `.xlsm` ファイル**（`.xlsx` と `.txt` は講師手元に保管）
- 受講生が `.xlsm` を開く際に「マクロを有効にする」ボタンを押す必要があります
- セキュリティの警告バーが出たら「コンテンツの有効化」をクリックするよう案内してください

---

## 受講生向け：VBE の開き方（当日の案内テキスト）

当日、受講生が VBE を開く際に口頭で案内する内容です：

```
① 配布された .xlsm ファイルを開く
② 「コンテンツの有効化」が出たらクリック
③ Alt + F11 で VBE を開く
④ 左側に「Module1」が見えていれば準備完了！
```

---

## 代替案：受講生自身でコードを貼り付ける場合（手順を教える）

`.xlsm` の事前準備が間に合わなかった場合、当日以下の手順を受講生に案内します：

```
【VBE へのコード貼り付け手順】

① デバッグ演習.xlsx を開く
② Alt + F11 → VBE が開く
③ 「挿入」→「標準モジュール」
④ VBAコード_バグあり.txt をメモ帳で開く
⑤ Ctrl + A → Ctrl + C でコピー
⑥ VBE の Module1 に Ctrl + V で貼り付け
⑦ F5 で実行（または Alt + F8 → マクロ選択 → 実行）
```

この手順に5〜10分かかることを想定してタイムスケジュールに余裕を持たせましょう。
