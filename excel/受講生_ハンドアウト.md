## 【1】デバッグ入門

### エラーの3種類

| 種類 | タイミング | 具体例 |
|---|---|---|
| **コンパイルエラー** | 実行前（文法ミス） | `End If` が抜けている |
| **実行時エラー** | 実行中 | 存在しないファイルを開こうとした |
| **論理エラー** | エラーは出ないが結果が間違い | `+` と `&` を混同した |

> **ポイント：** エラーメッセージは「敵」ではなく「ヒント」です。  
> 画面に出たエラー番号はそのままGoogleで検索できます。

---

### デバッグの3つの武器

#### 1. `Debug.Print` ─ イミディエイトウィンドウへの出力

```vba
Sub サンプル()
    Dim i As Long
    For i = 1 To 5
        Debug.Print "i の値：" & i   ' Ctrl+G でイミディエイトウィンドウを開く
    Next i
End Sub
```

- `MsgBox` との違い：処理が止まらない。ループ内での確認に最適。
- **使い方：** 確認したい箇所に書くだけ。確認が終わったらコメントアウトしておく。

#### 2. ブレークポイント（`F9`）

- 行番号の左側をクリック（または `F9`）で設定・解除。
- 設定した行で処理が一時停止する。
- 停止中にマウスオーバーすると変数の現在値が確認できる。

#### 3. ステップ実行（`F8`）

- `F8` を押すたびに1行ずつ実行される。
- コードを書いたらまず `F8` で動かす習慣をつけると、問題の早期発見になる。

## 【2】エラーハンドリング

### なぜエラー処理が必要か？

実務で動かすマクロは「ファイルが見つからない」「シートが存在しない」など  
想定外の状況で止まることがあります。  
エラー処理なしでは「処理途中のデータが残る」「ユーザーが混乱する」問題が起きます。

---

### `On Error GoTo` の基本パターン

```vba
Sub サンプル()

    On Error GoTo ErrorHandler   ' ① エラーが起きたらここへ飛ぶと宣言

    ' 通常の処理
    Workbooks.Open "C:\data.xlsx"

    Exit Sub                     ' ② 正常終了（ErrorHandler をスキップ）

ErrorHandler:                    ' ③ エラー発生時の処理
    MsgBox "エラー番号：" & Err.Number & vbCrLf _
         & "内容：" & Err.Description
    Application.ScreenUpdating = True   ' ④ 後片付けを忘れずに

End Sub
```

> **よくある落とし穴：** `Exit Sub` を忘れると、正常終了時も `ErrorHandler` が実行されてしまいます。

---

### `On Error Resume Next` の使いどころ

```vba
' 使ってよいケース：エラーが出ても問題ない単発処理
On Error Resume Next
Set ws = Worksheets("存在するかわからないシート名")
On Error GoTo 0   ' ← 必ずリセットする！

If ws Is Nothing Then
    MsgBox "シートが見つかりませんでした"
End If
```

> **注意：** ループ内で `On Error Resume Next` を使うと、すべてのエラーが無視されてしまいます。使う範囲は最小限にしましょう。

---

### `Err` オブジェクトの主なプロパティ

| プロパティ | 内容 | 使用例 |
|---|---|---|
| `Err.Number` | エラー番号（整数） | `Debug.Print Err.Number` |
| `Err.Description` | エラーの説明文 | `MsgBox Err.Description` |

## 【3】Function（関数）とコードの再利用

### Sub と Function の違い

| | `Sub` | `Function` |
|---|---|---|
| 戻り値 | なし | **あり** |
| 呼び出し方 | `Call プロシージャ名` | `変数 = 関数名(引数)` |
| 主な用途 | 一連の処理を実行する | 計算・判定結果を返す |

---

### Function の書き方

```vba
' ─── 定義 ───
Function 最終行取得(ws As Worksheet, col As Long) As Long
    '                ↑ 引数                         ↑ 戻り値の型
    最終行取得 = ws.Cells(ws.Rows.Count, col).End(xlUp).Row
    '  ↑ 関数名 = 戻り値  という形で返す
End Function

' ─── 呼び出し ───
Sub 集計()
    Dim lastRow As Long
    lastRow = 最終行取得(ActiveSheet, 1)   ' すっきり！
End Sub
```

---

### 引数の渡し方

```vba
' ByVal：値のコピーを渡す（元の変数は変わらない）← 基本はこちら
Function 計算(ByVal 数値 As Long) As Long

' ByRef：参照を渡す（関数の中で元の変数も変わる）← 意図しない変更に注意
Function 計算(ByRef 数値 As Long) As Long
```

> **迷ったら `ByVal` を使う** のが安全です。

---

### モジュール分割の考え方

```
標準モジュール
├── modMain    ← メインのマクロ（実行ロジック）
└── modUtil    ← 共通 Function（最終行取得・評価判定など）
```

共通処理を `modUtil` にまとめておくと、複数のマクロから呼び出せて保守しやすくなります。

## 【4】シートの高度な操作

### シートの参照方法

```vba
' パターン1：名前で指定（シート名変更に弱い）
Worksheets("集計")

' パターン2：インデックスで指定（順番変更に弱い）
Worksheets(1)

' パターン3：変数に入れて使う ← 最も安全
Dim ws As Worksheet
Set ws = Worksheets("集計")
ws.Cells(1, 1).Value = "タイトル"
```

> **重要：** オブジェクト変数には `Set` が必要です。

---

### シートの作成・削除

```vba
' シートを末尾に追加してシート名を設定
Worksheets.Add(After:=Worksheets(Worksheets.Count)).Name = "新シート"

' シートを削除（確認ダイアログを非表示にする）
Application.DisplayAlerts = False
Worksheets("削除するシート").Delete
Application.DisplayAlerts = True
```

---

### シートの存在確認（定番パターン）

```vba
Dim ws As Worksheet

On Error Resume Next
Set ws = Worksheets("確認するシート名")
On Error GoTo 0

If Not ws Is Nothing Then
    ws.Delete
    Set ws = Nothing
End If
```

---

### 全シートをループする

```vba
Dim ws As Worksheet

For Each ws In ThisWorkbook.Worksheets
    Debug.Print ws.Name   ' 全シートの名前を出力
Next ws
```

### パフォーマンス最適化の定番コード

```vba
Sub メインマクロ()

    ' ─── 処理前に停止 ───
    Application.ScreenUpdating  = False   ' 画面更新を止める
    Application.Calculation     = xlCalculationManual   ' 自動計算を止める
    Application.DisplayAlerts   = False   ' 確認ダイアログを非表示

    On Error GoTo ErrorHandler

    ' ─── 実際の処理 ───
    ' ...

    ' ─── 正常終了 ───
    Call 後片付け
    MsgBox "完了しました", vbInformation
    Exit Sub

ErrorHandler:
    MsgBox "エラー：" & Err.Description, vbCritical
    Call 後片付け

End Sub

Sub 後片付け()
    Application.ScreenUpdating  = True
    Application.Calculation     = xlCalculationAutomatic
    Application.DisplayAlerts   = True
End Sub
```
