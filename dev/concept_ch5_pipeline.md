## 不均衡データと Precision/Recall ── 乳がんデータで考える

```
【乳がんデータの特徴】

  全データ：569件
  ├── 良性（benign）：357件（62.7%）
  └── 悪性（malignant）：212件（37.3%）

  → ワインデータ（3クラス均等）とは異なり、クラスの比率が偏っている
  → 正解率だけでは「良性に偏って判定する」モデルが高スコアになる危険がある
```

```
【医療診断では Recall が最重要】

  「悪性を良性と診断してしまう（見逃し）」のコストは非常に高い
      ↓ 治療が遅れて命に関わる可能性
  「良性を悪性と診断してしまう（誤報）」のコストは低い
      ↓ 再検査・精密検査につながるだけ

  したがって：Recall（再現率）を最重視する場面
  → Recall が高いモデル ＝ 悪性を見逃さないモデル が求められる
```

> 📌 **Ch.3 で学んだ Precision/Recall の考え方がここで実務と直結します。**
> 「指標をどう読むか・何を重視するか」を自分で判断できることが今日の到達目標です。
