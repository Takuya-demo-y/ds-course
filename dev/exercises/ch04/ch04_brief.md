## よくあるエラーと対処

| エラー | 原因 | 対処 |
|-------|------|------|
| `ValueError: Input contains NaN` | `dropna()` を忘れている | `df = df.dropna()` を実行 |
| 配列の長さが合わない | `df` の行数と `y_pred` の長さが不一致 | `print(df.shape)` で確認 |
| グラフが表示されない | `plt.show()` を忘れている | セル末尾に `plt.show()` を追加 |
| `AttributeError` | インポートを忘れている | 最初のセルを再実行 |
