## データサイエンティストの仕事フロー

```
① データ収集・準備
        ↓
② グラフで全体像を把握（EDA）  ← Ch.1〜2
        ↓
③ 「この変数が重要そう」と仮説を立てる
        ↓
④ 機械学習モデルで仮説を検証   ← Ch.3〜4
        ↓
⑤ 結果を解釈して言葉にする・報告する ← 各Chの「考察」
```

| 作業 | 実務での時間比率（目安） |
|------|----------------------|
| データ収集・確認・クリーニング | 約 40〜50% |
| EDA・グラフで仮説を立てる | 約 20〜30% |
| モデル構築・評価 | 約 10〜20% |
| 結果の解釈・報告 | 約 10〜20% |

> 📌 **モデルを「作ること」より、データを理解して結果を解釈する方が圧倒的に重要です。**

## AI（Copilot）の使い方ルール

```
【今日の役割分担】

  人間がやること               AI（Copilot）に任せること
  ────────────────            ────────────────────────
  「何を分析するか決める」      「コードを書く」
  「グラフから仮説を立てる」    「ライブラリの使い方を調べる」
  「結果を言葉にする」          「エラーを修正する」

```

### AI への質問の型（5点セット）

```
【知りたいこと】〇〇を調べたい / 〇〇を計算したい

【環境】Python 3.8.6、JupyterLab
【困っていること】どう書けばよいかわからない
```

## 今日の学習スタイル と JupyterLab 操作

```
【3層構造で学ぶ】

① 理論（なぜ）  ─── 座学スライドで学ぶ
        ↓
② 実装（どうやって）  ─── Copilot を使いながら自分でコードを書く
        ↓
③ 検証（どうなったか）  ─── 気づきメモ・考察セクションを書く（AI禁止）
```

### JupyterLab の基本操作

| 操作 | 方法 |
|------|------|
| **セルを実行する（最重要）** | **Shift + Enter** |
| セルを選択する | セルをクリック |
| 新しいセルを追加 | 上部メニュー「＋」ボタン |

| よくあるトラブル | 対処法 |
|--------------|--------|
| 実行しても何も起きない | **Shift + Enter** を押す（クリックだけでは動かない） |
| `NameError` が出る | 上のセルから順番に実行し直す |
| `[*]` が消えない | 「Kernel」→「Restart Kernel」で再起動 |

## 動作確認をしましょう

`exercises/ch01/ch1_data_analysis_student.ipynb` を JupyterLab で開き、
**最初のセル（準備セル）** を実行してください。

```python
import pandas as pd
from sklearn.datasets import load_wine

wine = load_wine()
df   = pd.DataFrame(wine.data, columns=wine.feature_names)
df["品種名"] = ["wine_" + str(t) for t in wine.target]
print(f"行数: {len(df)}  列数: {len(df.columns)}")
```

✅ `行数: 178  列数: 14` と表示されれば **準備完了**です！
